# Gaming

## Descripción general

Este proyecto consiste en una página web estática desarrollada en HTML y CSS que presenta información sobre el mundo de los videojuegos. Incluye secciones sobre desarrolladoras importantes como Nintendo y Rockstar Games, junto con algunos de sus títulos más destacados.

El objetivo principal es aplicar buenas prácticas de estructura HTML semántica y estilos CSS externos para mejorar la presentación visual y la experiencia del usuario.

---

## Mejoras realizadas

### Acciones correctivas

* Se organizó la estructura HTML utilizando etiquetas semánticas como `header`, `article`, `aside` y `footer`.
* Se eliminaron estilos en línea para trabajar únicamente con CSS externo.
* Se corrigieron tamaños y atributos de imágenes para evitar deformaciones.
* Se mejoró la legibilidad del contenido con tipografía adecuada.

---

### Acciones evolutivas

* Se aplicaron estilos globales para mejorar la apariencia general del sitio (tipografía, colores y espaciado).
* Se diferenciaron visualmente las secciones principales (`article`) y secundarias (`aside`).
* Se añadieron efectos `hover` para generar interactividad en imágenes y enlaces.
* Se mejoró el menú de navegación con colores, alineación centrada y transiciones suaves.
* Se optimizó la presentación de imágenes haciéndolas responsivas y centradas.

---

## Resultado

El sitio final presenta una estructura clara, diseño moderno y una mejor experiencia visual para el usuario, cumpliendo con los estándares básicos de desarrollo web frontend.
